<h1 align="center">AlleBlockV2</h1>
<p align="center">Najświeższy zestaw filtrów dla uBO/AdGuard usuwający lub dezaktywujący zdecydowaną większość zbędnych, a także irytujących elementów portali ogłoszeniowych Allegro, OLX i innych. Kontynuacja projektu <a href="https://github.com/maciejtarmas/AlleBlock"><i>AlleBlock</i></a> od @maciejtarmas.</p>
<p align="center">Brak negatywnego wpływu na podstawową funkcjonalność platform - lista została stworzona dla zwiększenia komfortu użytkowania.</p>
<p align="center">Do skutecznego działania listy zalecana jest wtyczka uBlock Origin lub AdGuard; wbudowany bloker reklam przeglądarki Brave również powinien sobie poradzić.</p>
<p align="center">Jeśli zauważyłeś/aś element do ukrycia/zablokowania, bądź gdy filtry nie działają poprawnie, <a href="https://github.com/CrusheerPL/AlleBlockV2/issues">daj znać - za każde zgłoszenie serdecznie dziękuję.</a></p>

<h5>* - prace nad wdrożeniem w toku.</h5>

<hr>

<p align="center">The freshest uBO/AdGuard filter list which aims to improve the user experience of Allegro, OLX, etc. platforms by hiding or deactivating vastly most of their annoyances. Derived from the @maciejtarmas' <a href="https://github.com/maciejtarmas/AlleBlock"><i>AlleBlock</i> project</a>.</p>
<p align="center">No negative impact on basic sites' functionality.</p>
<p align="center">For best performance, the uBlock Origin or AdGuard plugin is strongly recommended. Also, the built-in Brave Browser's ad-blocker can be used.</p>
<p align="center">Have an element to hide/block, or something does not work properly? <a href="https://github.com/CrusheerPL/AlleBlockV2/issues">The <i>Issues</i> section is yours! Thanks a lot for your help.</a></p>
